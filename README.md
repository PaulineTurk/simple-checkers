# Simple Checkers

Jeu de dames classique 8x8 en 2 joueurs - interface minimaliste accessible depuis mobile.

## Règles implémentées

- Déplacement en diagonale d'une case en avant
- Capture obligatoire quand disponible
- Multi-capture (manger plusieurs pièces d'un coup)
- Promotion en dame à la dernière ligne
- Dame se déplaçant librement en diagonale (avant/arrière)

## Installation

```bash
npm install
```

## Tests

```bash
npm test
```

## Jouer

Ouvrez `index.html` dans un navigateur web.
